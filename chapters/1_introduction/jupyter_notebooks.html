Jupyter Notebooks¶

Jupyter notebooks are interactive environments where you can write and run Python code.

Code Cells¶

# This is a code cell
# Select this code cell by clicking on it
# Run this code cell by either clicking the 'Run button' above
# or using the shortcut 'Shift-Enter'

Welcome!

Every time we run a code block, Python will execute any code written, and if there’s an output, it will print it after each code block.

# If there are more than 1 line of python code within a code cell
# Running the code cell, will output the latest line of code

Comments¶

# You might have noticed that some text in our code block 
# are shaded greenish-gray and are preceded by a # symbol
# When you run this code block, you should see nothing happening.

These shaded words are comments.

Comments are instructions ignored by our Python interpreter. Just like a real life notebook, you are encouraged to write down personal learning notes in comments as we go along.

# If you forgot to add '#'
# Python assumes you giving it coding instructions
For example, running this code block should throw an error.

  File "<ipython-input-4-13824d4f2c20>", line 3
    For example, running this code block should throw an error.
              ^
SyntaxError: invalid syntax

Errors¶

Speaking of errors, as we continue to practice our programming, we’ll be meeting a lot of these. Remember, these errors are a very common experience for any programmer so don’t fret!

Errors give us instant feedback on our code. Often (but not always), Python is even smart enough to identify which portion of our code is causing the error.

print 'What is wrong with this code?'

  File "<ipython-input-5-3e181984ee25>", line 1
    print 'What is wrong with this code?'
                                        ^
SyntaxError: Missing parentheses in call to 'print'. Did you mean print('What is wrong with this code?')?
